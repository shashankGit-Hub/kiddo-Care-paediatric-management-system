# 📌 Project Overview

Kiddo Care is a web-based hospital management system designed specifically for pediatric healthcare. It streamlines patient registration, visit tracking, vaccination management, and medical record maintenance through an easy-to-use interface.

---

# ✨ Features

## 👶 Patient Management

* Register new patients
* Edit patient information
* Search patients
* View patient profiles

## 🩺 Visit Management

* Record patient visits
* Diagnosis management
* Prescription management
* Visit history

## 💉 Vaccination Module

* Vaccination records
* Vaccination schedule
* Track completed vaccinations

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/shashankGit-Hub/kiddo-Care-paediatric-management-system.git
```

Move into the project

```bash
cd kiddo-Care-paediatric-management-system
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the virtual environment

Windows

```bash
venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Apply migrations

```bash
python manage.py migrate
```

Start the development server

```bash
python manage.py runserver
```

Open your browser and visit

```
http://127.0.0.1:8000/
```
